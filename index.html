Workshop de Unity

30/04/2021

Pré-Requisitos

  1. Instalar Unity Hub
  2. No Unity Hub, criar uma conta pessoal (grátis)
  3. Em Preferences>License Management ativar uma nova licença
  4. Instalar a versão 2020.2.x do Unity
  5. (Opcional) Todos os módulos podem ser desativados, incluindo o Visual Studio, uma vez que não serão necessários para esta workshop
  6. Em Projects clicar em New para começar um novo projeto. Na lista de templates escolher o 3D.

Interface

De uma forma geral, a interface do Unity está dividida em 4 áreas:

  1. Scene View - visão do mundo
  2. Hierarchical View - lista de objectos atualmente no mundo
  3. Inpector View - propriedades do objecto selecionado
  4. Project View - ficheiros do projeto (assets)

Navegação e Manipulação

Focus

Duplo clique na Hierarchical View - Câmara posiciona-se de forma a focar o objecto.

Zoom

Roda do rato na Scene View - Câmara aproxima-se ou afasta-se.

Move

Botão do meio do rato na Scene View - Câmara move-se para os lados.

Orbit

Alt + Botão esquerdo do rato na Scene View - Câmara roda em torno de um ponto previamente focado.

First Person

Botão direito do rato + WASD na Scene View - Câmara move-se como se fosse um FPS, o rato controla a orientação e as teclas o movimento.

Manipulação

Existem 3 formas básicas de manipular um objecto no mundo: translação, rotação e escala. Cada uma tem uma ferramenta específica para o efeito, com determinados eixos que podem ser ajustados com o rato.

É possível alterar estes vetores diretamente na componente Transform do objecto, inserindo um novo valor ou ajustando o existente com o rato. Para limpar os valores existe a opção de Reset.

Base

06:00

  1. Botão direito do rato na Hierarchical View e escolher 3D Object>Cube. Colocar o cubo na posição (0, 0, 0) e escala (10, 0.1, 10).
  1. Descarregar o pacote de assets desta workshop: workshop.unitypackage
  2. Abrir o pacote e importar todos os ficheiros.
  1. No Project View abrir a pasta WorkshopAssets>Car e arrastar o carro para dentro da Scene View, de forma a ficar em cima do chão.
  1. Carregar no botão Play para iniciar o jogo. As teclas para controlar o carro são WASD, ou as teclas direcionais.
    2. ATENÇÃO: Alterações aos objetos da cena feitas durante o jogo são descartadas quando se termina.

Texturas e Materiais

08:00

  1. Botão direito do rato no Project View e escolher Create>Material.
  2. Na propriedade Albedo escolher a cor pretendida para o carro.
  3. Arrastar o Material para cima do carro para o atribuir.
  1. Usando o editor de imagem preferido (Paint por exemplo), criar uma imagem com resolução de 1024x1024px, e desenhar a pista.

    2. Esta imagem deve ser guardada na pasta Assets do projeto. Para a abrir rapidamente esta pasta pode-se usar o botão direito do rato no Project View e escolher Show in Explorer.

  1. Botão direito do rato no Project View e escolher Create>Material.
  2. Arrastar o Material para cima do chão para o atribuir.
  3. Arrastar a imagem criada para o espaço em branco à esquerda da propriedade Albedo.
  1. Escalar o chão para que o tamanho da pista fique grande o suficiente para o carro mover-se. Será algo à volta de (100, 0.1, 100).

    2. ATENÇÃO: A escala do carro não deve ser alterada.

Câmara

10:00

  1. Na Hierarchical View arrastar a câmara Main Camera para cima do carro.

    2. ATENÇÃO: A câmara deverá ser o ultimo filho dentro do carro, senão o Unity dará um erro de prefab.

  2. Posicionar a câmara de forma a permitir guiar o carro. Usar a preview que aparece no canto direito para visualizar como vai ficar. Uma posição que funciona bem é (0, 2.5, -5) e rotação (18, 0, 0).
  1. Ajustar a posição do carro, para iniciar dentro da pista.
  1. No Project View abrir a pasta WorkshopAssets>Track e adicionar saltos e obstáculos para tornar a pista mais interessante.

Play!

Animação

07:00

  1. Botão direito do rato na Hierarchical View e escolher 3D Object>Create Empty. Colocar na posição (0, 0, 0).
  2. Botão direito do rato no objecto criado anteriormente e escolher 3D Object>Cube. Colocar na posição (0, 1, 0).
  3. No Project View abrir a pasta WorkshopAssets>Explosion e arrastar a Explosion para cima do objecto criado no ponto 1. Colocar na posição (0, 1, 0).
  1. Selecionar o cubo criado no ponto 2 e abrir a tab Animation. Caso esta tab não esteja visível, ir a Window>Animation>Animation (Ctrl+6).
  2. Clicar em Create e escolher o nome e localização para o ficheiro de animação que será criado.
  3. Clicar no botão redondo vermelho, que inicia o modo de gravação.
  4. Botão direito do rato na propriedade Rotation do cubo e escolher Add Key.
  5. Na linha do tempo, ir para a posição 1 segundo.
  6. Botão direito do rato na propriedade Rotation do cubo e escolher Add Key. Mudar a rotação para (0, 360, 0).
  7. Clicar novamente no botão redondo vermelho para terminar o modo de gravação.

Play!

Colisões e Scripts

05:00

  1. Selecionar o cubo criado na secção anterior e, nas propriedades dele, ativar a propriedade Is Trigger que se encontra na componente Box Collider.
  2. Selecionar a explosão adicionada na secção anterior e, nas propriedades dela, desativar na primeira checkbox, ao lado esquerdo do nome. Nota que o nome do objecto na Hierarchical View muda para uma cor mais clara.
  1. Botão direito do rato no Project View e escolher Create>C# Script.
  1. Duplo clique no script criado para abrir o editor predefinido (pode ser o Notepad), e colocar o seguinte codigo dentro da classe, a seguir ao método Update.
    2. 
public GameObject explosionPrefab;

void OnTriggerEnter(Collider other){
	explosionPrefab.SetActive(true); 
	gameObject.SetActive(false);
}
  1. No Project View, arrastar o script criado para cima do cubo.

    2. ATENÇÃO: Se o script tiver algum erro, não será possível associar. Na consola é apresentada uma mensagem a vermelho com a causa do erro.

  2. Selecionar o cubo e, nas propriedades dele, procurar a componente que corresponde ao script adicionado. Arrastar a explosão para a propriedade Explosion Prefab.

Play!

Prefabs

05:00

  1. Na Hierarchical View arrastar o objecto pai, que contém o cubo e explosão, para o Project View. Nota que a cor do nome desse objecto (e filhos) passa de preto a azul.
  2. No Project View arrastar o prefab acado de criar para a pista. Repetindo a gosto.

Play!

UI e Compilação

05:00

  1. Botão direito do rato na Hierarchical View e escolher UI>Text. Escrever na propriedade Text de componente Text o nome do participante.
  1. No menu ir a File>Build Settings....
  2. Clicar em Add Open Scenes.
  3. Clicar em Build e escolher a pasta onde vai ser gerado o executável.
